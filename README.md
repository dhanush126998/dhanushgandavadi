# Dhanush Gandavadi Academic Website

This version uses a fully white background and separate pages for every navigation topic.

## Pages
- `index.html` — Home
- `about.html` — About
- `research.html` — Research
- `publications.html` — Publications
- `awards.html` — Awards
- `contact.html` — Contact

## Customize
1. Replace `assets/profile-placeholder.jpg` with your photo.
2. Add your CV as `assets/Dhanush_Gandavadi_CV.pdf`.
3. Update email and profile links in `contact.html`.
4. Add final citations and DOI links in `publications.html`.
5. Open `index.html` to preview.
